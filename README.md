# AWS Provisioning Demo (Python + Mock Boto3)

This demo simulates AWS resource provisioning using Python.  
It reads from `resources.json` and “creates” EC2, S3, and IAM resources with fake boto3 calls.

### Features
- Python script (`provisioner.py`) loads infrastructure-as-code JSON.
- Mock AWS SDK simulates provisioning events.
- HTML/JS frontend displays the simulation output in real-time.

### Technologies
Python · boto3 (mocked) · JSON · HTML · JavaScript
